# python-to-mermaid

Generate mermaid diagrams from python code.

## Usage

```python
from python_to_mermaid import generate_mermaid

generate_mermaid(code)
```

## Installation

```bash
pip install python-to-mermaid
```

## Development

```bash
uv sync
```

## Testing

```bash
pytest tests/ -v
```

## Publishing

```bash
./publish.sh
```

```bash
chmod +x publish.sh
```
